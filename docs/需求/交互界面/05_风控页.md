# 界面需求：风控页（Risk Management）

## 页面路径
`/risk-management`

## 功能描述

展示和管理所有风控机器人。包括余额监控、价格监控、余额定时报告、Telegram 日常提醒和报警。用于自动化监控账户安全和价格异常，并在异常时通过 Email/Telegram 发送通知。

│ 金额变动提醒间隔(min):      │

### 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| Email 1/2/3 | 输入框 | 空 | 接收通知的邮箱地址（最多3个） |
| 金额定时提醒间隔 | 数字 | 20 | 每隔多少分钟发送一次余额报告 |
| 金额变动提醒间隔 | 数字 | 20 | 余额变化后每隔多少分钟提醒一次 |
| Balance 警戒值 | 数字 | 100 | USDT 余额低于此值时触发报警 |
| Token 警戒值 | 数字 | 10000 | Token 数量低于此值时触发报警 |
| 启动模式 | 切换 | 立即启动 | 立即启动 / 定时启动 |

## 状态与交互

- **开关**：切换时启停风控机器人
- **配置参数**：点击弹出 Modal 配置通知参数
- **保存**：保存配置后自动启动（如启动模式为立即启动）
- **通知发送**：通过 Email SMTP 或 Telegram Bot API 发送

## 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/risk/alert` | POST | 保存风控告警配置 |
| `/api/risk/start` | POST | 启动风控机器人 |
| `/api/risk/stop` | POST | 停止风控机器人 |

## 设计要点

- 页面布局与"机器人部署"一致，3列卡片网格
- 风控机器人卡片使用盾牌/盾牌图标
- 弹窗表单纵向排列，字段标签左对齐
- 输入框：深色背景（#2d2d3d），白色文字
- 保存按钮：蓝色主按钮
