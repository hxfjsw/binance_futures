# 界面需求：用户管理页（User Management）

## 页面路径
- `/user/settings` → 用户-后台设置
- `/user/logs` → 用户-日志管理

## 功能描述

用户管理模块包含两个子页面：后台设置（用户列表管理）和日志管理（登录审计）。管理员可以查看和管理用户账户，以及查看所有登录记录。

## 交互元素清单

### 用户-后台设置

| 元素 | 说明 |
|------|------|
| 表格 | 用户列表 |
| 复选框 | 全选/单选用户 |
| 账户列 | 用户邮箱账号 |
| 身份列 | 用户角色（admin / user） |
| 员工名称列 | 员工姓名（可选） |
| 操作按钮 | 编辑 / 删除 / 禁用 |

### 用户-日志管理

| 元素 | 说明 |
|------|------|
| 表格 | 登录日志列表 |
| 账户列 | 登录账号 |
| 登录时间列 | 登录时间（2026/6/19 13:45:40） |
| 登录IP列 | 登录来源IP |
| 筛选条件 | 日期范围 / 账户 / IP |
| 导出按钮 | 导出日志到 CSV/Excel |

## 状态与交互

- **侧边栏展开**：点击"用户"菜单展开子菜单（后台设置 / 日志管理）
- **子菜单高亮**：当前所在子页面高亮显示
- **表格空数据**：显示 "No data available"
- **表格分页**：支持分页和每页条数选择
- **筛选条件**：支持按日期范围、账户名、IP地址筛选

## 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/user/list` | GET | 用户列表 |
| `/api/user/logs` | GET | 登录日志 |
| `/api/user/update` | POST | 更新用户信息 |
| `/api/user/delete` | POST | 删除用户 |

---

## 设计要点

- 表格样式：深色表头，交替行颜色（可选），hover 高亮
- 复选框：表头复选框全选，行复选框单选
- 分页器：底部居中，显示总条数和页码
- 筛选器：表格上方，日期范围选择器 + 输入框
- 侧边栏展开动画：200ms ease-in-out
