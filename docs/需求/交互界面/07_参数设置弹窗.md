# 界面需求：参数设置弹窗（Global Settings Modal）

## 触发方式

点击 TopBar 上的 **"参数设置"** 按钮（`docs/截图/12_screenshot_20260619_135649.954.png`）

## 功能描述

配置当前选中账户的交易参数，包括保证金模式（全仓/逐仓）、持仓模式（单向/双向）、杠杆倍数。支持全局应用到所有账户。

## 交互元素清单

| 元素 | 类型 | 说明 |
|------|------|------|
| 账户 | 文本（只读） | 当前登录账户 |
| 标签 | Select | 选择要配置的账户标签（账户1, 账户2...） |
| 市场 | 文本（只读） | 当前选中的交易对（如 TRXUSDT） |
| 全局账户更改 | Switch | 关闭/开启，开启后修改所有账户 |
| 保证金模式 | ToggleButton | 全仓 / 逐仓（二选一） |
| 持仓模式 | ToggleButton | 单向持仓 / 双向持仓（二选一） |
| 杠杆倍数 | InputNumber | 1~125 的整数 |
| 取消按钮 | Button | 关闭弹窗，不保存 |
| 确认按钮 | Button | 保存配置并应用到账户（蓝色主按钮） |

## 状态与交互

- **标签下拉**：选择不同账户标签，切换当前配置对象
- **全局账户更改**：开启后，所有账户统一使用相同参数
- **保证金模式**：切换时会影响订单的保证金计算方式
- **持仓模式**：切换时会影响是否可以同时持有多空仓位
- **杠杆倍数**：修改后需要确认，可能影响现有持仓的强平价格
- **确认按钮**：点击后发送 `POST /api/account/settings`，成功后显示 Toast 提示

## 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/openapi/getTradeSettings` | GET | 获取当前账户的交易设置 |
| `/api/account/settings` | POST | 保存账户参数设置 |

## 设计要点

- 弹窗宽度：约 500px，居中显示
- 背景遮罩：半透明黑色（rgba(0,0,0,0.6)）
- 关闭按钮：右上角 X 图标，红色 hover
- 表单字段：标签左对齐，输入框右对齐
- 切换按钮：蓝色选中状态，灰色未选中状态
- 提示文字：黄色小字（全局账户更改的说明）
