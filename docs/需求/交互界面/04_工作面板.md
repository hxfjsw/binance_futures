# 界面需求：工作面板（Trading Desk）

## 页面路径
`/work-panel`

## 功能描述

提供手动交易干预能力和实时行情监控。包含K线图、实时盘口深度、以及手动对敲/批量挂单/集群挂单/持仓/委托管理等功能。是机器人策略的"手动补充"操作台。

## 交互元素清单

### K线图区域

| 元素 | 说明 |
|------|------|
| 交易对标题 | TRXUSDT |
| O/H/L/C/V | 开盘价/最高价/最低价/收盘价/成交量 |
| 时间轴 | 10:00 ~ 14:30（5分钟间隔） |
| 价格轴 | 右侧显示价格刻度 |
| 蜡烛图 | 绿色涨，红色跌 |
| 成交量 | 底部柱状图 |
| 刷新 | 定时刷新（5秒间隔） |

### 交易操作区（Tabs）

#### Tab 1: 手动对敲

| 元素 | 说明 |
|------|------|
| 交易量输入 | 输入总交易量 |
| 分笔数输入 | 输入分几笔成交 |
| 随机值按钮 | 每笔交易量随机分配 |
| 中间值按钮 | 每笔交易量取平均值 |
| 对敲按钮 | 执行对敲交易（主按钮，蓝色） |
| 结果表格 | 时间 / 价格 / 数量 / 分笔数 |

#### Tab 2: 批量挂单

| 元素 | 说明 |
|------|------|
| 订单列表 | 批量添加/删除订单行 |
| 每行：交易对 | 方向 | 类型 | 价格 | 数量 |
| 批量提交按钮 | 一次性提交所有订单 |
| 批量取消按钮 | 取消未成交订单 |

#### Tab 3: 集群挂单

| 元素 | 说明 |
|------|------|
| 账户选择 | 多选多个账户 |
| 统一参数 | 价格/数量/方向 |
| 集群提交 | 在选中的账户上同时挂单 |

#### Tab 4: 持仓

| 元素 | 说明 |
|------|------|
| 持仓列表 | 交易对 / 方向 / 数量 / 均价 / 未实现盈亏 |
| 平仓按钮 | 一键平仓 |

#### Tab 5: 当前委托

| 元素 | 说明 |
|------|------|
| 委托列表 | 时间 / 交易对 / 方向 / 价格 / 数量 / 状态 |
| 撤单按钮 | 单笔撤单 / 批量撤单 |

#### Tab 6: 历史委托

| 元素 | 说明 |
|------|------|
| 历史列表 | 时间 / 交易对 / 方向 / 价格 / 数量 / 状态 |
| 筛选条件 | 日期范围 / 交易对 / 状态 |

### 右侧固定面板：实时盘口深度

| 元素 | 说明 |
|------|------|
| 卖盘（红色） | 卖1~卖N：价格从高到低，数量 |
| 买盘（绿色） | 买1~买N：价格从低到高，数量 |
| 当前价格 | 中间显示最新成交价 |
| 市价/可用/价格/数量/委托 | 底部操作快捷栏 |
| 只显示 | 筛选显示档位数量 |

- 卖盘价格：红色字体
- 买盘价格：绿色字体
- 数量：灰色字体
- 最新价格：高亮显示（绿色背景 + 白色文字）

## 状态与交互

- **Tab 切换**：点击切换不同功能面板，内容区切换，URL不变
- **对敲操作**：输入交易量 + 分笔数 → 点击对敲 → 显示结果表格
- **盘口深度**：实时推送（WebSocket），每档数量变化时高亮闪烁
- **账户选择器**：右上角下拉选择当前操作账户（账户1 ~ 账户N）

## 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/openapi/klines` | GET | K线数据 |
| `/api/openapi/depth` | GET | 实时盘口深度 |
| `/api/openapi/trades` | GET | 成交记录 |
| `/api/openapi/openOrders` | GET | 当前委托 |
| `/api/openapi/historyOrders` | GET | 历史委托 |
| `/api/openapi/getAllPositions` | GET | 所有持仓 |
| `/api/trade/match` | POST | 手动对敲 |
| `/api/trade/batch` | POST | 批量挂单 |
| `/api/trade/place` | POST | 下单 |
| `/api/trade/cancel` | POST | 撤单 |

## 设计要点

- K线图：使用 TradingView Lightweight Charts，占页面宽度约 70%
- 右侧盘口深度：固定宽度约 200px，独立滚动
- 交易操作区：Tabs 下方，占页面高度约 40%
- 表格：空数据时显示 "No data available"
- 颜色：卖盘红色 #ef4444，买盘绿色 #22c55e
- 实时更新：盘口深度每 1 秒刷新，K 线图每 5 秒刷新
