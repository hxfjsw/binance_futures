# 界面需求：机器人配置弹窗（Robot Config Modal）

## 触发方式

点击机器人卡片上的 **"配置参数"** 或 **"查看"** 按钮。

## 功能描述

为每个机器人配置运行参数。不同机器人类型的参数字段不同。弹窗根据机器人类型动态渲染表单字段。

## 弹窗结构（通用框架）

```
┌────────────────────────────────────┐
│ 机器人名称                    [X]    │
├────────────────────────────────────┤
│ 动态表单字段（根据机器人类型变化）      │
│                                     │
│ 价格模式: 一般模式 | 安全模式          │
│ 启动模式: 立即启动 | 定时启动          │
│                                     │
│ 机器人描述:                         │
│ [文本域]                            │
│                                     │
│              [保存] 或 [关闭]        │
└────────────────────────────────────┘
```

---

## 通用字段（所有机器人）

| 字段 | 类型 | 说明 |
|------|------|------|
| 价格模式 | ToggleButton | 一般模式 / 安全模式 |
| 启动模式 | ToggleButton | 立即启动 / 定时启动 |
| 机器人描述 | TextArea | 自定义备注说明 |
| 保存按钮 | Button | 保存配置并关闭弹窗 |
| 关闭按钮 | Button | 关闭弹窗（仅查看模式） |

## 交互元素

### 范围输入（Range Input）

```
┌────────────┐  ~  ┌────────────┐
│ 最小值      │     │ 最大值      │
└────────────┘     └────────────┘
```

- 两个 InputNumber 输入框，中间用 "~" 分隔
- 最小值必须小于最大值，否则校验失败

### 切换按钮（ToggleButton）

```
┌──────────┬──────────┐
│  选中项    │  未选中项  │
│  蓝色背景  │  灰色背景  │
└──────────┴──────────┘
```

- 二选一或三选一，蓝色表示选中，灰色表示未选中
- 点击切换选中状态

### 提示信息（Alert）

- 黄色背景 + 黄色文字
- 用于显示重要警告（如 AMM 的"不可出入金"提示）

---

## 状态与交互

- **配置模式**（点击"配置参数"）：所有字段可编辑，显示"保存"按钮
- **查看模式**（点击"查看"）：所有字段只读，显示"关闭"按钮
- **表单校验**：提交时校验所有必填字段，错误时显示红色提示
- **保存成功**：Toast 提示"保存成功"，弹窗关闭
- **保存失败**：显示错误信息（如参数范围不合理）

## 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/robot/config` | POST | 保存机器人配置 |
| `/api/robot/config` | GET | 获取机器人配置（查看模式） |

---

## 设计要点

- 弹窗宽度：约 500px，内容多时支持内部滚动
- 表单字段纵向排列，标签左对齐，输入框右对齐
- 范围输入：两个输入框并排，中间波浪号
- 切换按钮：横向排列，圆角，选中状态蓝色背景
- 提示信息：黄色警告框，带图标（可选）
- 底部按钮：右侧对齐，取消在左，保存在右（蓝色主按钮）
