# 界面需求：登录页（Login）

## 页面路径
`/`（根路径）

## 功能描述
用户登录系统的入口页面。支持账户密码登录、语言切换、用户协议确认、忘记密码。

## 交互元素清单

| 元素 | 类型 | 说明 |
|------|------|------|
| 账户名输入框 | Input | placeholder="请输入账户名"，邮箱格式验证 |
| 密码输入框 | Input.Password | placeholder="请输入密码"，带眼睛图标切换显示/隐藏 |
| 密码强度指示 | 文本 | 显示密码强度值（0~100） |
| 用户协议复选框 | Checkbox | 必须勾选才能登录 |
| 用户协议链接 | Link | 点击弹窗显示用户协议全文 |
| 登录按钮 | Button | 主按钮（蓝色），disabled 状态到勾选协议且输入有效 |
| 语言切换 | Button | 中英文切换（ENGLISH） |
| 忘记密码 | Link | 跳转或弹窗重置密码 |

## 状态与交互

- **登录按钮**：未勾选协议时 disabled，账户/密码为空时 disabled
- **密码强度**：实时显示，8位以上、含大小写+数字+符号时显示为强
- **登录失败**：错误提示（账户或密码错误）
- **登录成功**：存储 JWT Token，重定向到 `/dashboard`

## 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/auth/login` | POST | `{account, password}` |

## 设计要点

- 全屏背景：深色渐变（#1a1a2e → #16213e）
- 左面板：占约 40% 宽度，居中偏左
- 右面板：占约 60% 宽度，金融插画（矢量图）
- 底部：黄色免责声明小字，固定底部
